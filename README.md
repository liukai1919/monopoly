# 🍁 大富翁 · 加拿大版

一款给全家人在客厅一起玩的 Monopoly:**电视当棋盘,手机当手柄**。
局域网直连,不需要外网,不需要注册,扫码就能玩。

- 🖥️ 大屏(电视/电脑)显示棋盘、骰子动画、事件日志、玩家资产
- 🏙️ 开局可选择熟悉的经典棋盘，或会随市场与行业景气变化的「城市脉搏」棋盘
- 📱 每人用手机浏览器扫码加入,掷骰子、买地、盖房、竞拍、谈交易
- 🤖 人不够可以加最多 4 个 AI 补位(2–6 人)
- 📜 经典完整规则:买地 / 收租 / 盖房与酒店 / 机会与宝箱卡 / 监狱 / 抵押赎回 / 拍卖 / 自由交易 / 破产清算
- 🇨🇦 棋盘为加拿大城市主题(温哥华、多伦多压轴),界面中文
- 🔌 手机锁屏/掉线自动重连,游戏不中断

## 快速开始(家庭游戏夜)

1. **双击 `start.bat`**(首次运行会自动装依赖、构建前端,之后秒开)
2. 浏览器会自动打开 `http://localhost:3000` —— 把这个窗口投到电视/大屏,点「创建游戏」
3. 家人手机连**同一个 Wi-Fi**,扫大屏上的二维码,输入名字加入
4. 人不够点「添加 AI」,然后大屏上点「开始游戏」

> ⚠️ 第一次启动时 Windows 防火墙会弹窗询问,请勾选**「专用网络」并允许**,
> 否则手机连不上。如果错过了弹窗:Windows 安全中心 → 防火墙 → 允许应用通过防火墙 → 勾选 Node.js;
> 或以管理员运行:
> `netsh advfirewall firewall add rule name="Monopoly" dir=in action=allow protocol=TCP localport=3000`

## 可选房规(开局前在大屏上选择)

| 设置 | 说明 |
|---|---|
| 棋盘体验 | 城市脉搏（大厅默认）或经典棋盘；只改变视觉，不改变规则 |
| 免费停车奖池 | 税款进奖池,踩中「免费停车」全拿(默认关) |
| 行业景气 | 每轮最强行业的相关地块租金 +50%（默认开） |
| 时长限制 | 200/400 手后按净资产结算,适合控制游戏时间(默认玩到只剩一人) |

## 常用命令(开发)

```bash
npm install        # 安装依赖
npm run dev        # 开发模式 (server:3000 + vite:5173, 热更新)
npm run build      # 构建前端到 client/dist
npm start          # 生产模式: 单端口 3000, 托管前端
npm test           # 引擎单元测试 + 100 局 AI 自动对局模拟
npx tsx scripts/e2e.ts   # 端到端: 模拟手机玩家走真实 Socket 打完整局 (需先启动服务器)
```

## 架构

```
shared/   纯 TypeScript 游戏引擎 (零依赖)
          ├─ board.ts / cards.ts        加拿大版棋盘与卡牌数据 (改地名只动这里)
          ├─ engine/game.ts             applyAction(state, action) → 新状态 + 动画事件
          ├─ engine/helpers.ts          租金计算、建房/抵押校验 (前端按钮状态同款复用)
          └─ ai/simpleAi.ts             启发式 AI (买地/盖房/竞拍/还债/评估交易)
server/   Node + Express + Socket.IO
          ├─ 房间码 + 扫码加入 + UUID 断线重连
          ├─ 权威游戏状态, 每次动作全量广播
          └─ AI 驱动 (可配 AI_THINK_MS) 与拍卖 25 秒倒计时
client/   React + Vite, 一套代码两种视图
          ├─ /board/:code   大屏: 经典/城市脉搏双棋盘、共享棋子与特效层
          └─ /play/:code    手机: 按回合阶段渲染操作 (掷骰/购买/竞价/筹钱/交易)
```

服务器是唯一的状态权威,手机只发「意图」;引擎是纯函数,完整单元测试 +
100 局 AI 完整对局模拟保证规则正确性(拍卖、破产清算、均衡建造等都有覆盖)。
